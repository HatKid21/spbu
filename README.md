# Spbu Modern Programming Methods
This repository contains my work for the Modern Programming Methods course at SPbU, which spans 2.5 semesters.

The course focuses on **C++ programming**, core programming concepts, and implementing fundamental data structures and algorithms from scratch.

## Projects completed so far

The repository currently includes the following projects, in order of study:
1) Standard program compilation + refactoring
2) Multi-level menu in terminal, using structures
3) Sorting algorithm realization
4) Custom long-number class
5) Custom vector and doubly-linked list realization

All projects are built using **CMake**.

## Topics covered

Within that course we learned how to:
- Compile and refactor programs in C++
- Use structures and classes
- Object-oriented programming
- Use of C++ standart template library (STD)
- Implementing basic data structures and algorighms
- Lot of theory
